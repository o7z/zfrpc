# Web 界面设计

```

│  zfrpc                          [设置] [退出] │
├────────┬─────────────────────────────────────┤
│        │                                     │

│        │                                     │
│ 仪表盘  │  ┌───────────────────────────────┐  │

│        │  │                               │  │

│        │                                     │
└────────┴─────────────────────────────────────┘
```

## 公共组件

### Modal

基于 `<dialog>` + `showModal()` 的公共弹窗组件（`src/lib/components/Modal.svelte`）。

- Props: `title`（标题）、`open`（双向绑定控制显隐）、`width`（最大宽度，默认 520px）
- 点击遮罩层关闭
- 自动居中，半透明遮罩背景
- 用于设置弹窗、删除确认弹窗等所有对话框场景

### Switch

开关组件（`src/lib/components/Switch.svelte`），用于布尔选项切换。

## 页面详情

### 仪表盘

- frpc 运行状态（运行中/已停止/异常）
- 当前配置概览
- 快速操作（启动/停止/重启）
- 最近日志（最后 10 条）

### 配置管理

- 左侧：配置文件列表（名称、路径、活跃状态）
- 右侧主区域：表单编辑器
  - 顶部操作栏：删除按钮（红色）+ 保存按钮（蓝色），横向排列
  - 服务器配置：可折叠，含传输/日志子配置
  - 代理列表：每项含可折叠子板块（域名配置、带宽与协议、负载均衡、健康检查、元数据），默认折叠
  - 访问者列表
- 右侧预览区：TOML 预览，鼠标移入显示复制按钮
- 删除配置：Modal 二次确认弹窗

### 设置

- 使用 Modal 弹窗展示
- frpc 程序路径（含文件浏览器和路径验证）
- 工作目录
- 保存按钮

## 认证

- 可选密码保护
- HttpOnly Cookie Session
- 7 天 TTL
