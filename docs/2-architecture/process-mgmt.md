# frpc 进程管理

## 进程生命周期

```
┌─────────┐    ┌─────────┐    ┌─────────┐
│  空闲    │───▶│  启动    │───▶│  运行    │
└─────────┘    └─────────┘    └─────────┘
     ▲                              │
     │         ┌─────────┐          │
     └────────│  停止    │◀─────────┘
              └─────────┘
```

## 管理功能

| 功能 | 实现方式 |
|------|----------|
| 启动 | `child_process.spawn('frpc', ['-c', configPath])` |
| 停止 | `process.kill()` + 优雅退出信号 |
| 重启 | 停止 → 等待退出 → 启动 |
| 状态监控 | 监听 `exit`/`error`/`stdout`/`stderr` |
| 日志捕获 | 流式读取 stdout/stderr，WebSocket 推送 |

## 配置发现

启动时自动扫描：
1. 当前目录 `frpc.toml` / `frpc.ini`
2. 常见路径：`/etc/frp/`, `~/.frp/`
3. 用户自定义路径（通过 Web 界面添加）
